# Calculator Project (Task 24.2.4)

## Постановка задачи
В рамках задания необходимо создать простой калькулятор и написать к нему тесты.
Калькулятор должен выполнять базовые арифметические операции, а для каждого метода
нужно написать по одному позитивному тесту.

## Описание проекта
В проекте реализован класс Calculator с методами сложения, вычитания, умножения и деления.
Для каждого метода написан отдельный тест, который проверяет корректную работу калькулятора.
Проект разбит на папки, чтобы было удобно работать с кодом и тестами.

## Используемые инструменты
Проект написан на языке Python.
Для тестирования используется библиотека unittest.
Также рекомендуется использовать pytest версии 6.2.5–7.4.4.

## Как запустить тесты
Тесты можно запустить из корня проекта командой:
pytest

Также тесты можно запустить с помощью файла:
python run_tests.py

## Пример результата
При успешном запуске тестов все проверки проходят без ошибок,
что означает корректную работу методов калькулятора.

## Итог
В проекте реализован простой калькулятор и написаны позитивные тесты для каждого метода.
Проект загружен на GitHub и соответствует требованиям задания 24.2.4.
